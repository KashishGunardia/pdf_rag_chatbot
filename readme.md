# PDF RAG Chatbot

This project is a Retrieval Augmented Generation (RAG) chatbot that allows users to upload a PDF and ask questions about its content.

## Tech Stack
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq (LLaMA 3)

## Installation

pip install -r requirements.txt

## Run the App

streamlit run app.py
